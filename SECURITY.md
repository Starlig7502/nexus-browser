# Security Policy — Nexus Browser

## Supported Versions
We actively monitor and patch security vulnerabilities for the main production branch.

| Version | Supported |
| ------- | --------- |
| 1.0.x   | ✅ Yes    |
| < 1.0   | ❌ No     |

## Reporting a Vulnerability
Because Nexus Browser handles sensitive data like passwords and web sessions, we take security very seriously. If you find a vulnerability (e.g., a bypass in our 64-thread downloader or a flaw in the `scan_extension_code` sandbox), **please do not open a public GitHub issue.**

Instead, please report security vulnerabilities by:
1. Opening a **Draft Security Advisory** directly on this GitHub repository under the "Security" tab.
2. Clearly describing the exploit vector, steps to reproduce, and potential impact.

We will evaluate your report within 48 hours and coordinate a security fix before publicly disclosing the vulnerability. Thank you for keeping Nexus secure!
